键盘快捷键 (Keyboard shortcuts)

Keyboard shortcuts (键盘快捷键) 说明 Description
Ctrl + P 切换预览 Switch watch/unwatch
Ctrl + H 插入标题(H3) Insert heading 1~6
Ctrl + A / Command + A 全选 Select all
Ctrl + B 插入粗体 Insert bold
Ctrl + E 插入Emoji符号 Insert :emoji:
Ctrl + F / Command + F 查找/搜索 Start searching
Ctrl + G / Command + G 切换到下一个搜索结果项 Find next search results
Ctrl + I 插入斜体 Insert italic
Ctrl + K 插入行内代码 Insert inline code
Ctrl + L 插入链接 Insert link
Ctrl + U 插入无序列表 Insert unordered list
Ctrl + Z / Command + Z 撤销 Undo
Ctrl + Y / Command + Y 重做 Redo
Ctrl + G 插入图片 Insert image ![]()
Ctrl + O 插入有序列表 Insert ordered list
Ctrl + K 插入引用 Insert blockquotes
Ctrl + ? 打开使用帮助对话框 Open help dialog
